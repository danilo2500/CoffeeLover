# ☕ CoffeeLover

A SwiftUI app for discovering and saving coffee images. Swipe right to like, left to dislike.

## Features
- Tinder-style swipe interface with visual feedback
- Save favorites with SwiftData persistence
- Image caching with Kingfisher
- unit test coverage
- UI test coverage

## Tech Stack
SwiftUI • SwiftData • Async/Await • MVVM • Dependency Injection

## Setup
git clone https://github.com/yourusername/CoffeeLover.git
open CoffeeLover.xcodeprojRun with `⌘ + R` • Test with `⌘ + U`
